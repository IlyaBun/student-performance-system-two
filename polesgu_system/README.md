# 🎓 Информационно-аналитическая система оценки успеваемости студентов инженерного факультета ПолесГУ

## 📋 Описание
Программная система для количественной и качественной оценки успеваемости студентов инженерного факультета Полесского государственного университета (ПолесГУ).

### Основные возможности:
- **Ролевая модель**: Администратор, Преподаватель, Студент
- **Учет оценок**: Поддержка 10-балльной шкалы и зачетов
- **Виды контроля**: Лекции, Лабораторные, Практические, Экзамены, Зачеты, Курсовые
- **Аналитика**: 
  - Расчет среднего балла, успеваемости, качества знаний
  - Тепловая карта проблемных дисциплин
  - Прогнозирование риска отсева
  - Сравнительный анализ групп
- **Кэширование**: Мгновенная загрузка дашборда
- **Расширенное логирование**: Отслеживание всех изменений (было/стало)
- **Современный GUI**: Темная тема по умолчанию, адаптивный интерфейс на customtkinter

## 🚀 Быстрый старт

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2. Запуск системы
```bash
python main.py
```

### 3. Вход в систему
При первом запуске создается база данных с демо-данными.

**Администратор по умолчанию:**
- Логин: `admin`
- Пароль: `RwQNt`

**Тестовые пользователи:**
- Преподаватели: `teacher1`, `teacher2` (пароль: `password`)
- Студенты: `student1` ... `student250` (пароль: `password`)

## 👥 Роли пользователей

### Администратор (admin)
- Полный доступ ко всем вкладкам
- Управление пользователями (повышение ролей)
- Просмотр системных логов
- Настройки системы

### Преподаватель (teacher)
- Просмотр всех студентов и групп
- Добавление/редактирование оценок
- Аналитика и отчеты
- Настройки профиля

### Студент (student)
- Просмотр только своих оценок
- Личная статистика
- Настройки профиля

## 📊 Функционал вкладок

1. **Главная (Dashboard)**: KPI метрики, тепловая карта, группа риска, прогноз отсева
2. **Студенты**: Реестр студентов с фильтрацией и поиском
3. **Дисциплины**: Список предметов с кафедрами и преподавателями
4. **Журнал оценок**: Ввод оценок с указанием вида контроля
5. **Аналитика**: Рейтинги, сравнение групп, детальная статистика
6. **Пользователи** (только админ): Управление доступом
7. **Настройки**: Тема, язык, профиль

## 🔧 Технические особенности

- **СУБД**: SQLite с нормализованной схемой
- **GUI**: customtkinter (темная тема по умолчанию)
- **Графики**: matplotlib
- **Безопасность**: Хеширование паролей bcrypt, параметризированные SQL-запросы
- **Кэширование**: In-memory кэш с инвалидацией по событиям
- **Логирование**: Детальный аудит действий

## 📝 Примечания

- Система автоматически создает базу данных при первом запуске
- Все новые пользователи регистрируются как студенты
- Темная тема установлена по умолчанию (рекомендуется для длительной работы)
- Кэш аналитики обновляется автоматически при изменении оценок
